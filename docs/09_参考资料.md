## 6. 你需要重点看什么

### 如果你先做平地自主版

- 重点看 Nav2
- 重点看 OpenAI 的 Responses / Realtime / STT / TTS
- 重点看供电、安全保护、定位与避障

### 如果你做跨楼层版

- 重点看 Jetson Orin Nano
- 重点看能爬楼梯的平台案例
- 重点看特种履带与安全设计
- 重点看跨楼层任务如何拆分为“能到达”与“能安全到达”

## 7. 工程判断说明

本项目方案里有几个关键判断：

- `ESP32 + Jetson + ROS2 + 大模型代理` 比单板全包更稳
- `先做平地自主，再评估跨楼层` 比一步到位更容易成功
- `Unity 数字孪生先行` 非常适合你的能力背景

这些判断来自机器人常见工程实践，以及上述官方/社区资料的综合推断。
